class pyidf.fans.FanComponentModel¶

Bases: pyidf.helper.DataObject

Corresponds to IDD object Fan:ComponentModel A detailed fan type for constant-air-volume (CAV) and variable-air-volume (VAV) systems. It includes inputs that describe the air-distribution system as well as the fan, drive belt (if used), motor, and variable-frequency-drive (if used).

class pyidf.fans.FanPerformanceNightVentilation¶

Bases: pyidf.helper.DataObject

Corresponds to IDD object FanPerformance:NightVentilation Specifies an alternate set of performance parameters for a fan. These alternate parameters are used when a system manager (such as AvailabilityManager:NightVentilation) sets a specified flow rate. May be used with any type of fan except not with Fan:ComponentModel. If the fan model senses that a fixed flow rate has been set, it will use these alternate performance parameters. It is assumed that the fan will run at a fixed speed in the alternate mode.
