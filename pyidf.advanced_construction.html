class pyidf.advanced_construction.SurfaceConvectionAlgorithmInsideAdaptiveModelSelections¶

Bases: pyidf.helper.DataObject

Corresponds to IDD object SurfaceConvectionAlgorithm:Inside:AdaptiveModelSelections Options to change the individual convection model equations for dynamic selection when using AdaptiveConvectiongAlgorithm This object is only needed to make changes to the default model selections for any or all of the surface categories. This object is for the inside face, the side of the surface facing a thermal zone.
