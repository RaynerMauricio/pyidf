number_of_vertices¶

field Number of Vertices

shown with 120 vertex coordinates – extensible object “extensible” – duplicate last set of x,y,z coordinates (last 3 fields), remembering to remove ; from “inner” fields. for clarity in any error messages, renumber the fields as well. (and changing z terminator to a comma ”,” for all but last one which needs a semi-colon ”;”) vertices are given in GlobalGeometryRules coordinates – if relative, all surface coordinates are “relative” to the Zone Origin. If world, then building and zone origins are used for some internal calculations, but all coordinates are given in an “absolute” system. Default value: “autocalculate” value >= 3.0
